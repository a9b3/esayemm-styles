# esayemm-styles

Style reset.

## Installation

Install using yarn

```sh
yarn add esayemm-styles --save
```

## Usage

Import wherever you have your global css. This package depends on your project
being built by webpack.

in js

```js
import 'esayemm-styles';
```

or in your css

```css
@import '~esayemm-styles';
```
